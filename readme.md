## 四、課程內容 Course Description

### 整體敘述 Overall Description

本專題介紹美國憲法裁判中的解釋方法及其相關問題。

第一部分（三週）「導論」：泛論憲法解釋所涉基本問題。  
第二部分（三週）「憲法解釋理論」：探討各種憲法解釋正當化理論。  
第三部分（四週）「憲法解釋方法」：介紹憲法解釋所使用之方法。  
第四部分（三週）「指標案例比較」：比較美國聯邦最高法院、德國聯邦憲法法院等就特定議題（如同婚、墮胎）之憲法裁判及其解釋方法。  
最後（三週）由選課同學提出研究報告並進行評論。

修課同學須：

- 每週課前完成指定閱讀  
- 輪流擔任導讀  
- 期末提出符合學術規範之研究報告  
- 進行同儕評論並修正定稿
